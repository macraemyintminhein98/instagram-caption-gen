# InstaCaption AI

InstaCaption AI is a simple web application designed to help you generate engaging Instagram captions effortlessly. Just input your desired keywords, select a tone, add some emojis, and let the AI craft the perfect caption for your posts.

## Features:

*   **Keyword-based Caption Generation:** Provide relevant keywords to guide the AI.
*   **Tone Selection:** Choose from various tones like friendly, funny, inspiring, or professional.
*   **Emoji Integration:** Easily include emojis to add flair to your captions.
*   **Instant Results:** Get your captivating captions in seconds.

## How to Use:

1.  Enter your desired keywords (e.g., "travel, adventure, sunset").
2.  Select the tone for your caption from the dropdown menu.
3.  (Optional) Add specific emojis you'd like to include.
4.  Click "Generate Caption" to see the AI-powered suggestion.
5.  Click "Get Your Unlimited Access!" to purchase the service.

## Technologies:

*   **Frontend:** HTML, Tailwind CSS, Vanilla JavaScript
*   **Backend (Conceptual):** Next.js/React (for a full build), LLM API (e.g., OpenAI, Hugging Face)
*   **Payments (Conceptual):** Stripe Checkout
